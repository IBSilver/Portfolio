University Life Survival

Newsgame about the housing crisis in Spain

2024 / Designer, Developer

Description

This project consists of creating a narrative newsgame using Twine based on news about a current issue in Spain.

The project was a 3 member group assignment for the 'Serious games' subject in 4th year of the Degree in Video Game Design and Development at CITM - UPC in Terrassa, Spain.

In this project I was in charge of the research, design and development of the game.

This project was done in Twine (Harlowe format) along with some basic HTML and CSS.

Task summary

  • Narrative design.
  • Development in Twine.
  • UI deisgn & implementation.
