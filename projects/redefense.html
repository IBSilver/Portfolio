Reefense

Serious game prototype about recycling

2025 / Designer, Developer

Game Design Document

Description

This project consists of choosing a topic, preparing appropriate documentation for a serious game (empathy map, stakeholders map, market study, etc.), and creating a simple prototype from scratch. Our group chose the topic of recycling.

The project was a 3 member group assignment for the 'Serious games' subject in 4th year of the Degree in Video Game Design and Development at CITM - UPC in Terrassa, Spain.

In this project I was in charge of the research, design and development of the game, more specifically enemies, containers, HUD and dialogues.

This project was made in Unity using free assets.

Task summary

  • Serious game documentation.
  • GDD organization.
  • Narrative design.
  • Enemies implementation.
  • Containers implementation.
  • HUD implementation.
  • Dialogues implementation.
