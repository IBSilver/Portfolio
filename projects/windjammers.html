Tribute to Windjammers (1994)

Recreation of an arcade video game

2022 / Developer

Description

This project consists on recreating the 1994 arcade game Windjammers, which was originally developed and published by Data East on the Neo Geo arcade system. The goal of this project was to pay tribute to the original game by replicating it.

The project was a 5 member group assignment for the 'Project I' subject in 1st year of the Degree in Video Game Design and Development at CITM - UPC in Terrassa, Spain.

In this project I was in charge of the compilation of animations from the original game, sprite editing, creation of spritesheets and implementation of animations in the game, both in the intro menus and characters.

This project was made using C and SDL.

Task summary

  • Documentation.
  • Sprite editing & spritesheets.
  • Menus & characters animations.
